Pypsy.signal.analysis
=====================

The :py:mod:`Pypsy.signal.filter` module provides most of the signal processing
and analysis functionality required by the :py:class:`~Pypsy.signal.Signal`
class and its subclasses, especially for EDA signal decomposition.
